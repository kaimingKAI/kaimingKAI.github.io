Chrome 常用快捷键

笔记本触控板不太好用，想要多用快捷键来替代光标操作。选了几个最常用的记录一下。

打开新标签页

    ctrl + t

切换到下一个标签页

    ctrl + tab

关闭当前标签页

    ctrl + w 或 ctrl + F4

打开新窗口

    ctrl + n

打开Chrome菜单

    alt + f

在当前页面搜索

    ctrl + f

跳转到地址栏

    ctrl + l
