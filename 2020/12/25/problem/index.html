Hexo遇到的问题汇总

汇总我使用Hexo时遇到的问题与解决方案，解决方案多数来自CSDN社区的帖子。
目前收录的问题有：

  • 无法加载主题
  • Github Page地址中有重复字段
  • Hexo不识别md语法的标题
  • 如何隐藏主页文章内容，只显示部分预览
  • can not read a block mapping entry

部署Github后无法加载主题

原因在于博客目录下 _config.yml 文件中URL与ROOT配置错误。

解决方案： 将_config.yml文件中URL：后修改为Github仓库链接。注意复制粘贴后删除.io后的斜杠 “/”
。

url：https://yourname.github.io

将URL下面的ROOT：后改为Github仓库的名称。注意名称前后加斜杠 “/”。

root:/yourname.github.io/

Github Page地址中有重复字段

出现类似于yourname.github.io/yourname.io 的地址
原因在于只有yourname.github.io中yourname与Github用户名完全一致时，才会被认定为个人页面，地址才会只有yourname.github.io。其他情形下皆会出现重复字段。

解决方案： 命名时将仓库yourname.github.io中的yourname与Github用户名完全一致。

Hexo不识别md语法的标题

解决方案： 在标题##标识后方添加空格，Hexo无法识别 “##标题”，必须“## 标题”。

如何隐藏主页文章内容，只显示部分预览

在撰写md文件时，在想要展示的预览后添加标识

<!--more-->

标识前的文本默认会在主页显示，表示后的需点击Read More才会显示。

can not read a block mapping entry

撰写md文件时有语法错误，常见的问题是缺少空格
我当时的问题是 

type:"categories"

正确写法须在：后加空格

type: "categories"
